About the Prison Yard

The classes consist of a wide variety of exercises, from bodyweight routines to weight training. Each workout is different, and customized for the participants.

We have all kinds of fun stuff for workout stations. We have tires that need to be flipped, sledgehammers that need to be thrown, and a few other fun things (Did someone say sled?).

You're going to work out for the full 50 minutes. You're going to be breathing hard at the end. You might want to call your mommy. Some people call it hard. We call it progress.
